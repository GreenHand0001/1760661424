# 项目简介

## 核心功能
- 基于 YOLOv8 的道路损害检测系统，支持识别 6 类道路损害：横向裂缝、网状裂缝、斜向裂缝、纵向裂缝、坑洞和修复区域
- 支持多种输入源检测：单张图片、图片文件夹、视频文件和本地摄像头
- 提供用户界面，实时显示目标位置、目标总数和置信度信息
- 支持自定义检测参数：置信度阈值和 IOU 阈值
- 支持批量检测功能，可直接查看所有检测结果
- 支持检测结果保存功能

## 适用场景
- 道路维护与检测
- 基础设施巡检
- 城市道路安全评估
- 道路损害自动识别与分析

## 技术特点
- 采用 Python 3.9 开发，依赖 opencv-python、PyQt5、ultralytics 和 torch1.9 等技术栈
- 提供完整的标注数据集（2000 张图片）
- 包含训练好的模型及训练结果
- 配有详细的项目运行教程
- 支持多种开发环境：Pycharm+Anaconda 或 VSCode+Anaconda
